Sales Analytics Dashboard Using Excel

Project Overview

This project focuses on analyzing sales data and building an interactive dashboard using Microsoft Excel. The objective of the project is to transform raw transactional sales data into meaningful business insights that help understand sales performance, profitability trends, and customer behavior.The dashboard provides a clear overview of sales performance across different regions, product categories, sales channels, and payment methods.

Project Goals

1. The main goals of this project were to:
2. Analyze sales performance across different regions
3. Understand profitability across product categories
4. Identify monthly sales and profit trends
5. Evaluate customer behavior (new vs returning customers)
6. Compare sales channels and payment methods
7. Build an interactive dashboard that summarizes key business insights

Data Preparation

Data cleaning and transformation were performed using Power Query. During this stage, several calculated columns were created to improve analysis:
Sales (Final Sales) – calculated revenue after discount
Profit – difference between total sales and product cost
Month – extracted from the sales date to analyze monthly trends

These transformations allowed better aggregation and visualization of sales data.

Key Performance Indicators (KPIs)

The dashboard tracks the following key metrics:
1. Total Sales
2. Total Profit

These KPIs provide a quick overview of overall business performance.

Dashboard Insights

The dashboard includes the following visual analyses:

1. Sales by Region – identifies top-performing geographic areas

2. Sales Channel Analysis – compares Online vs Retail performance

3. Sales and Profit by Product Category – evaluates profitability across product groups

4. Monthly Sales and Profit Trend – shows how business performance changes over time

5. Customer Type Analysis – compares new and returning customer contributions

6. Sales and Profit by Payment Method – analyzes customer payment preferences

Tools and Technologies Used: Microsoft Excel, Power Query, Pivot Tables, Pivot Charts, Data Visualization Techniques
